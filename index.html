Welcome!

This is Fire Bird Dashboard, designed and developed by Time Bremer and Arturo MB, makeover by, Technologx. This dashboard is designed for users to be able to get their weather icons out there for everyone to use. You can also use this dashboard to get wallpapers you've created out there.

You can download a sample of Fire Bird from Google Play, and get the source from GitHub.

If you find any bugs, please report them via the GitHub issue tracker. If you are a coder, it would be even better if you could contribute!
